# Projects

Nectar SDK is the shared foundation Black Bee Drones uses for competition missions. Each
repository holds the full mission README and code; this page lists the events and official
links only.

!!! note "SDK lineage"

    The SDK evolved over three generations: `tadinisdk` (ROS 1) → `mirela_sdk` (ROS 2) →
    **`nectar`**. Older projects reference the earlier names.
