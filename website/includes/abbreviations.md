*[FCU]: Flight Control Unit — the autopilot firmware (ArduPilot, PX4, …) on the vehicle
*[SITL]: Software In The Loop — autopilot firmware running as a desktop simulation
*[EKF]: Extended Kalman Filter — fuses GPS, vision, IMU, and other sensors into a pose estimate
*[VSLAM]: Visual Simultaneous Localization and Mapping
*[VIO]: Visual-Inertial Odometry
*[MAVROS]: MAVLink ↔ ROS 2 bridge package (`mavros`)
*[MAVLink]: Lightweight messaging protocol between companion computer and flight controller
*[uXRCE-DDS]: Micro XRCE-DDS — PX4 native DDS transport (UDP agent + `px4_msgs`)
*[RTL]: Return To Launch — fly back to the takeoff/home position
*[L4T]: Linux for Tegra — NVIDIA Jetson board support package
*[RSUSB]: RealSense USB user-space backend (`librealsense` built with RSUSB/libuvc)
*[mAP]: mean Average Precision — object-detection accuracy metric
*[NMS]: Non-Maximum Suppression — merges overlapping detection boxes
*[WBF]: Weighted Boxes Fusion — ensemble post-processing for detections
*[NMM]: Non-Maximum Merging — alternative box-merge strategy in the detection pipeline
*[PID]: Proportional–Integral–Derivative controller
*[GUI]: Graphical User Interface — the Qt6 desktop app (`ros2 run nectar app.py`)
